# Smart Home Security System

## Môn học

## Mô tả
Hệ thống bảo mật nhà thông minh sử dụng ESP32-CAM, PIR, MQTT và AI nhận diện khuôn mặt.

## Thành viên
- TV1: ESP32-CAM và phần cứng
- TV2: AI xử lý ảnh
- TV3: MQTT và Dashboard
- TV4: Voice Control, OTA và tích hợp

## Luồng hoạt động
PIR → ESP32-CAM → MQTT → AI → Dashboard → Cảnh báo

## Công nghệ sử dụng

- MQTT
- Python
- OpenCV
- face_recognition
